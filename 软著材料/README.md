## 您还需自行准备

### 1. 申请表个人信息
在 `软件著作权登记申请表.md` 中补充：
- 申请人名称（个人姓名或企业全称）
- 证件号码（身份证号或统一社会信用代码）
- 通讯地址、邮编
- 联系电话、邮箱
- 开发完成日期
- 首次发表日期（如未发表填"未发表"）

### 2. 身份证明文件
- **个人申请：** 身份证正反面复印件
- **企业申请：** 营业执照副本复印件（加盖公章）

### 3. 源代码文档打印
将 `源代码/前30页_源代码.txt` 和 `源代码/后30页_源代码.txt` 分别打印：
- **纸张规格：** A4纸
- **格式要求：** 每页50行代码，页眉含软件名称和页码
- **打印份数：** 1份
- **装订方式：** 左侧装订
- **页边距：** 上下各留空白

### 4. 软件说明书打印
将 `软件说明书.md` 转换为Word/PDF后打印：
- 包含软件界面截图（如有，建议加上）
- A4纸打印，1份

### 5. 如需代理机构
- 授权委托书（代理机构提供模板）

## 提交方式

### 线上提交（推荐）
1. 访问中国版权保护中心：https://www.ccopyright.com.cn
2. 注册账号 → 实名认证
3. 在线填写申请表
4. 上传源代码和说明书电子版
5. 邮寄纸质材料（部分情况）

### 线下提交
1. 所有材料打印并签字/盖章
2. 邮寄至：北京市西城区天桥南大街1号天桥艺术大厦A座 中国版权保护中心

## 申请费用

| 类型 | 费用 |
|------|------|
| 个人申请 | 免费（部分情况收审查费） |
| 企业申请 | 约 300-1000 元（视代理机构） |

## 办理周期

- 受理：提交后 1-7 个工作日
- 审查：受理后 30-60 个工作日
- 发证：审查通过后约 10 个工作日
- **总计约 2-3 个月**

## 常见问题

**Q: 源代码60页够吗？**
A: 标准要求首次提交60页（前30+后30）。如果审查员要求补充，可再提交。

**Q: 需要全部源代码吗？**
A: 不需要。提交代表性的源代码即可，但必须是本软件的原始代码。

**Q: 软件说明书需要截图吗？**
A: 强烈建议包含。每个功能模块配1-2张界面截图更有利于通过审查。

**Q: 可以用Markdown提交吗？**
A: 建议转换为PDF格式提交，Markdown仅作为源文件。

**Q: 软件名称可以修改吗？**
A: 可以，在申请表上填写你想要的名称即可。当前建议名称"灵集AI智能创作助手平台"。
