## 一、引言

### 1.1 编写目的

本说明书为"灵集AI智能创作助手平台"（以下简称"灵集AI"）的用户操作手册，旨在帮助用户和评审人员了解本软件的安装、功能、操作流程及使用场景。

### 1.2 软件概述

灵集AI是一站式智能创作助手平台，面向中文内容创作者，覆盖从**灵感采集 → AI创作 → 多平台发布 → 数据分析**的内容创作全流程。利用大语言模型（LLM）、视觉理解、语音合成、视频生成等AI技术，帮助用户发现热点、激发灵感、生成内容、一键发布。

### 1.3 运行环境

| 环境类型 | 配置要求 |
|---------|---------|
| **浏览器** | Chrome 90+, Safari 15+, Firefox 90+, Edge 90+ |
| **移动端** | iOS 15+ (Safari), Android 10+ (Chrome) |
| **原生应用** | iOS 15+ (Capacitor WebView), Android 10+ (Capacitor WebView) |
| **网络** | 建议 4G/WiFi 网络连接 |
| **屏幕** | 响应式设计，适配 320px - 1920px 宽度 |

## 三、功能模块详述

### 3.1 用户注册与登录

**功能描述：** 用户通过手机号+验证码方式进行注册和登录。

**操作步骤：**
1. 输入11位中国手机号
2. 点击「获取验证码」，完成图形验证后收到6位短信验证码
3. 输入验证码，可选填写用户名
4. 首次使用需选择账号类型（14种：创业品牌/知识IP/电商带货/B2B企业/个人博主/教培/餐饮/医美/房产/文旅/亲子/健身/宠物/家居），AI会根据账号类型推荐合适的创作方案

**用户界面：**
- 登录页面：手机号输入框 → 验证码输入框 → 图形验证码 → 登录/注册按钮
- 账号类型选择弹窗：14种类型卡片，单选确认

### 3.2 快速采集（AI对话助手）

**功能描述：** AI对话式灵感采集工具，支持多模态输入，是用户记录创意、收集素材的主要入口。位于底部导航栏中间"+"按钮。

**支持的输入方式：**

| 输入方式 | 功能说明 |
|---------|---------|
| **文字输入** | 直接输入想法、问题或素材内容 |
| **图片上传** | 上传图片文件，AI通过视觉模型理解图片内容并生成相关灵感 |
| **文档上传** | 支持 PDF、DOCX、TXT、MD 格式文件上传，AI自动解析文档内容 |
| **视频上传** | 上传 MP4/MOV 等视频文件（最大100MB），AI提取语音文字并分析视频内容 |
| **语音输入** | 按住录音按钮说话，上滑取消，系统实时显示语音识别文字结果 |
| **链接抓取** | 粘贴文章链接，AI通过 jina.ai Reader 自动抓取正文并分析提炼 |
| **联网搜索** | 开启「联网搜索」开关后，AI实时搜索互联网信息辅助回答 |

**AI模型选择：** 支持切换 Auto / DeepSeek / Qwen / Doubao 等多种模型。

**消息处理功能：**
- 每条AI回复可执行：复制、分享、编辑、删除、重新生成
- 可保存为灵感素材（存入灵感库）
- AI自动识别日程信息（日期、时间、标题等），生成日程卡片
- 文字润色：超过10个字的文本支持8种风格改写（小红书、公众号、抖音等）
- 生成图片和视频直接显示在对话中
- 支持创建、切换、重命名、删除多个聊天会话

### 3.3 灵感库

**功能描述：** 集中管理所有创作素材的管理中心，支持多媒体格式，提供丰富的筛选、排序、操作功能。

**内容类型：**
- 文字 — 文本灵感
- 图片 — 图片素材
- 视频 — 视频素材
- 音频 — 音频素材
- AI作品 — AI生成的内容

**筛选方式：**
- **类型筛选：** 全部 / 文字 / 图片 / 视频 / 音频 / AI作品
- **时间筛选：** 全部 / 今天 / 本周 / 本月 / 近3月
- **标签筛选：** 按自定义分类标签过滤
- **排序：** 最新/最旧、标题A-Z/Z-A

**操作功能：**
- 拖拽上传（支持图片、视频、音频、PDF、Word等格式）
- 批量选择、批量下载、批量删除
- 单条编辑（标题、正文、分类、标签）
- 图片/视频全屏预览
- 音频在线播放
- 收藏 / 归档 / 删除

**用户界面：**
- 灵感库列表页：网格或列表视图展示所有灵感卡片
- 每条卡片显示：类型图标、标题、摘要、时间、标签
- 详情页：完整内容 + AI分析结果 + 关联内容推荐

### 3.4 AI创作中心

**功能描述：** AI创作中心是本软件的核心功能模块，包含9大AI创作工具和创作方案推荐系统。

#### 3.4.1 创作方案（推荐组合）

根据用户选择的账号类型，AI自动推荐3-4套针对性的创作流水线。例如「灵感 → 文案 → 生图 → 发布」方案，点击即可一键启动多步骤自动创作。支持用户自定义组合创作流程。

**用户界面：**
- 顶部6步工作流进度条（灵感→文案→图片→数字人→视频→分发）
- 推荐方案卡片列表
- 各步骤自动携入上下文

#### 3.4.2 AI文案

**功能：** 多平台智能写稿工具。

**4步创作流程：**
1. **选材+输入：** 从灵感库选择素材 + 自由文字输入 + 链接粘贴自动解析 + 图片OCR识别
2. **选平台：** 8大平台选择（小红书/抖音/快手/公众号/微博/B站/知乎/短视频脚本）
3. **选风格：** 14种写作风格（感性走心/专业深度/营销转化/幽默段子等）
4. **选行业：** 25个行业专属模板（美妆/穿搭/美食/母婴/数码/家居/教育/职场/法律等），每个行业预设必含元素、避坑指南、开头钩子、行动号召

**特色功能：**
- 一次生成3个文案变体
- 「去AI味」开关让文案更自然
- 生成后可一键导入AI生图或AI视频

#### 3.4.3 AI生图

**功能：** 智能图片生成工具。

**8种预设类型：**
小红书封面 / 公众号头图 / 抖音封面 / 商品主图 / 壁纸 / 海报 / 社交配图 / 头像

每种预设自动匹配比例、风格、色调。支持批量生成（4张一批），AI智能提示词优化。高级设置支持种子值控制（0-21亿整数，可复现相同风格图片）、负面提示词。

生成后的图片可执行：重新生成/下载/存灵感库/图生视频/复制prompt。

#### 3.4.4 AI图片编辑

**功能：** 图片后期处理工具。
- 去除背景：AI一键抠图
- 画质增强：超分辨率放大
- 智能扩图：AI扩展图片边界

#### 3.4.5 AI配音（TTS）

**功能：** 文字转语音工具。

**特性：**
- 6种豆包TTS音色（男女声多种风格）
- 语速调节（0.5x - 2.0x）
- 音调调节（0.5 - 2.0）
- 文本上限2000字
- 支持浏览器中预览音频
- 支持下载MP3文件
- 声音克隆功能：上传30秒清晰人声样本，训练1-5分钟生成个人专属音色

#### 3.4.6 AI数字人

**功能：** 虚拟主播视频生成工具。

**7种工作模式：**
1. 一键生数字人：输入主题 → 全自动流水线（写稿→配音→上传→生成）
2. AI写稿：AI自动生成口播脚本（3个变体可选）
3. 批量生成：多个主题逐条处理
4. 多语言：支持中文/English/日本語/한국어
5. 用我的形象：上传角色头像+参考视频，通过wan2.2-animate模型让静态图复刻视频动作
6. 用我的分身：训练个人数字分身（HeyGen），之后可无限次生成口播视频
7. 手动配置：完全自定义每一步参数

**技术限制：** 音频时长上限20秒（wan2.2-s2v模型限制），系统自动拦截超长音频。

#### 3.4.7 AI视频

**功能：** 一键生成短视频工具。

**3步创作流程：**
1. **确定方向：** 选择首帧图片 + 风格预设 + 时长 + 主题
2. **分镜预览：** AI自动生成分镜脚本，可编辑每段的visualPrompt和字幕
3. **生成合并：** FFmpeg合并多段视频 + BGM + 字幕烧录合成最终成片

**特色功能：**
- 3个画质等级：流畅/标准/高清
- BGM智能推荐（6档风格）
- AI智能字幕（LLM改写为口播短句）
- 多首尾帧模式（支持5个中间关键帧）
- 一键成片模式（全自动生成）

#### 3.4.8 朋友圈九宫格

**功能：** 社交媒体3×3组图生成工具。

**6种场景预设：** 产品宣传 / 生活记录 / 节日纪念 / 摄影美学 / 创意排版 / 兴趣展示

每张封面配一句20字内朋友圈配文，支持一键打包ZIP（含9张封面+CSV标题清单）。

#### 3.4.9 多平台发布

**功能：** 一键多平台内容分发工具。

**已接入平台：** 微信公众号（OAuth）、微博（OAuth）

**发布流程：**
1. 编辑发布内容（标题、正文、封面图、标签）
2. 选择目标平台和账号（需先完成平台OAuth授权）
3. 选择立即发布或定时发布
4. 查看发布记录和状态（已发布/定时/草稿/失败）

对于抖音/小红书/视频号/B站等未接入API的平台，支持标记为「手动发布」并记录外部链接和互动数据。

### 3.5 热点监控

**功能描述：** 实时追踪全网热点，帮助用户发现创作选题。

**核心功能：**
- **雷达面板：** 数据概览（今日新增/累计热点/紧急/未读数）+ 来源分布（微博/知乎/B站/百度/抖音/头条/Bing/HackerNews 8大平台）

- **一键转灵感素材**

**监控词管理：**
- 添加/启用/停用/删除监控关键词
- 预设关键词库（多分类可选）
- 后台每3小时自动扫描各平台

**实时搜索：**
- 输入关键词，跨9大平台即时搜索
- 地域选择：全球+中国 / 仅中国 / 仅海外
- AI分析：相关度、重要性、要点、摘要

## 四、软件主要技术特点

### 4.1 AI能力总览

| AI能力 | 技术实现 | 功能说明 |
|--------|---------|---------|
| **文字生成** | DeepSeek / Qwen 大语言模型 | 文案写作、摘要提炼、内容改写、智能分析 |
| **图片理解** | 豆包视觉模型 | 图片内容识别、OCR文字提取 |
| **视频理解** | 视频ASR语音识别 + 内容分析 | 视频语音转文字、内容摘要 |
| **图片生成** | 豆包 Seedream 模型 | 多种风格和比例的AI图片生成 |
| **图片编辑** | AI图像处理 | 去背景、画质增强、智能扩图 |
| **视频生成** | Wan / Seedance 模型 | 文字转视频，自动字幕、BGM |
| **语音合成** | CosyVoice / Kokoro TTS | 多音色文字转语音 |
| **数字人** | wan2.2-s2v / HeyGen | 照片生成数字人形象 + 口播动画 |
| **联网搜索** | 聚合搜索（百度/Google/Bing） | 实时互联网信息检索 |
| **链接抓取** | jina.ai Reader | 网页正文提取与分析 |

### 4.2 技术架构

- **前端框架：** Next.js 14 (App Router) + React 18 + TypeScript + TailwindCSS 3.4
- **后端服务：** Next.js API Routes (BFF层) + Supabase (认证/数据库/存储)
- **AI服务：** 阿里云DashScope + 火山引擎豆包 + OpenRouter + HeyGen + jina.ai
- **数据库：** PostgreSQL (Supabase托管)
- **移动端：** Capacitor 8 混合应用框架
- **部署：** Vercel Serverless + 阿里云ECS

### 4.3 安全特性

- 全站HTTPS加密传输
- API密钥服务端统一管理，客户端不可见
- 用户数据隔离（数据库行级安全策略）
- 平台Token AES-256-GCM加密存储
- 点数系统原子扣点（PostgreSQL RPC），防并发超额消费

## 五、使用场景

1. **小红书博主：** 收藏爆款截图和链接 → AI生成种草文案 → AI生图配封面 → 一键发布
2. **短视频创作者：** 采集热点素材 → AI生成口播脚本 → AI数字人/视频生成 → 多平台发布
3. **公众号作者：** 收藏文章链接 → AI提炼素材 → AI生成大纲和深度内容 → 排版发布
4. **自媒体运营：** 批量整理素材 → 多平台文案生成 → 批量生图 → 定时发布 → 数据分析
5. **知识IP：** 声音克隆 + 数字分身 → AI写稿 → 批量生成口播短视频
