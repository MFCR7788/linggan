# 灵集AI 操作手册

## 产品概述

灵集AI 是一站式智能创作助手，面向中文内容创作者。从**灵感采集** → **AI创作** → **多平台发布** → **数据分析**，覆盖内容创作全流程。

**核心价值：** 用 AI 帮你发现热点、激发灵感、生成内容、一键发布，让创作更高效。

**访问方式：** H5 网页（zjsifan.com），同时提供 iOS/Android 原生应用。

---

## 快速开始

### 注册/登录

1. 输入 11 位中国手机号
2. 点击「获取验证码」，完成图形验证后收到 6 位短信验证码
3. 输入验证码，可选填写用户名
4. 点击登录/注册（新用户自动注册）

### 选择账号类型

首次使用时，系统会引导你选择账号类型（创业品牌 / 知识IP / 电商带货 / B2B企业 / 个人博主 / 教培 / 餐饮 / 医美 / 房产 / 文旅 / 亲子 / 健身 / 宠物 / 家居），AI 会根据账号类型推荐合适的创作方案。

## 五大核心模块

### 1. 快速采集（+号按钮）

AI 对话式灵感采集，支持多模态输入：

| 输入方式 | 说明 |
|---------|------|
| **文字输入** | 直接输入想法、问题、素材 |
| **图片上传** | 上传图片，AI 理解图片内容并生成相关灵感 |
| **文档上传** | 支持 PDF、DOCX、TXT、MD 文件 |
| **视频上传** | 上传 MP4/MOV 等视频（最大 100MB），AI 提取语音文字并分析 |
| **语音输入** | 按住说话，上滑取消，实时显示识别文字 |
| **链接抓取** | 粘贴文章链接，AI 自动抓取正文并分析 |
| **联网搜索** | 开启「联网搜索」开关，实时搜索互联网信息 |

**AI 模型选择：** 支持切换 Auto / DeepSeek / Qwen / Doubao 等多种模型。

**消息处理功能：**
- 每条 AI 回复可：复制、分享、编辑、删除、重新生成
- 可保存为灵感素材（存入灵感库）
- AI 自动识别日程信息（日期、时间、标题等），生成日程卡片
- 文字润色：超过 10 个字的文本支持 8 种风格改写（小红书、公众号、抖音等）
- 生成图片和视频直接显示在对话中

**多会话管理：** 支持创建、切换、重命名、删除多个聊天会话。

---

### 2. 灵感库

集中管理所有创作素材，支持多媒体格式。

**内容类型：** 文字 / 图片 / 视频 / 音频 / AI 作品

**筛选方式：**
- 类型筛选：全部 / 文字 / 图片 / 视频 / 音频 / AI 作品
- 时间筛选：全部 / 今天 / 本周 / 本月 / 近 3 月
- 标签筛选：按自定义分类标签
- 排序：最新/最旧、标题 A-Z/Z-A

**操作功能：**
- 拖拽上传（支持图片、视频、音频、PDF、Word 等）
- 批量选择、批量下载、批量删除
- 单条编辑（标题、正文、分类、标签）
- 图片/视频全屏预览
- 音频在线播放
- 收藏 / 归档 / 删除

### 3. AI 创作中心

#### 创作方案（推荐组合）

根据账号类型推荐自动化创作流水线。例如「灵感 → 文案 → 生图 → 发布」，一键启动多步骤自动创作。支持自定义组合。

#### 9 大 AI 创作工具

**AI 文案** — 多平台智能写稿
- 4 步流程：选素材 → 选平台 → 选风格 → 选行业
- 支持 8 个平台：小红书 / 抖音 / 快手 / 公众号 / 微博 / B站 / 知乎 / 短视频脚本
- 14 种风格：感性走心 / 专业深度 / 营销转化 / 幽默段子
- 25 个行业：有对应的必备要素、避坑指南、行动号召
- 多模态输入：灵感素材 + 自由文字 + 链接 + 图片 OCR
- 一次生成 3 个变体，支持「去 AI 味」

**AI 生图** — 智能图片生成
- 8 种预设：小红书封面 / 公众号头图 / 抖音封面 / 商品主图 / 壁纸 / 海报 / 社交配图 / 头像
- 自动匹配比例、风格、色调
- 支持批量生成（4 张一批）
- AI 智能提示词优化

**AI 图片编辑**
- 去除背景
- 画质增强
- 智能扩图

**AI 配音 (TTS)** — 文字转语音
- 多种音色选择
- 显示字符数和点数消耗
- 支持下载

**AI 数字人** — 虚拟主播视频
- 上传照片生成专属数字人形象
- AI 自动撰写口播脚本
- 自动生成带字幕、背景音乐的口播视频
- 可选 480P 或 720P

**AI 视频** — 一键生成短视频
- 文字转视频
- 3 档质量：快速 / 标准 / 高级
- AI 智能配乐推荐
- 自动字幕生成
- 支持分镜脚本生成

**九宫格** — 社交媒体组图
- 6 种场景预设
- 自动生成 3×3 朋友圈/小红书九宫格

**热点监控** — 发现创作选题（详见下方）

**多平台发布** — 一键多平台分发（详见下方）

**最近作品：** 展示所有 AI 生成的作品，可按类型查看。

### 4. 热点监控

实时追踪全网热点，帮助找到创作选题。

**雷达面板：**
- 数据概览：今日新增、累计热点、紧急、未读数
- 来源分布：微博 / 知乎 / B站 / 百度 / 抖音 / 头条 / Bing / HackerNews
- 筛选：监控词、来源平台、重要程度（紧急/高/中/低）、时间范围（1小时/今天/7天）
- 排序：时间 / 相关度 / 重要程度
- 每条热点包含：热度、平台、AI 摘要、相关度评分、创作建议、可信度评估、原文链接
- 一键转为灵感素材

**监控词管理：**
- 添加/启用/停用/删除监控关键词
- 预设关键词库（多分类可选）
- 每个关键词独立展示关联热点
- 后台每 3 小时自动扫描

**实时搜索：**
- 输入关键词，跨 9 大平台搜索
- 地域选择：全球+中国 / 仅中国 / 仅海外
- AI 分析结果：相关度、重要性、要点、摘要

---

### 5. 多平台发布

一键发布内容到多个平台。

**已接入平台：** 微信公众号、微博

**发布流程：**
1. 编辑内容（标题、正文、封面图、标签）
2. 选择目标平台和账号（需先 OAuth 授权）
3. 立即发布或定时发布
4. 查看发布记录和状态（已发布/定时/草稿/失败）

**手动发布标记：** 对于未接入 API 的平台，可标记为「手动发布」并记录外部链接和互动数据。

**发布数据追踪：** 自动（微信公众号/微博API）或手动录入各平台阅读量、点赞、评论等数据。

---

### 6. 数据洞察

- **时间范围：** 7 天 / 30 天 / 90 天
- **概览数据：** 总发布数、浏览量、点赞、评论、分享、收藏、总互动、平均互动
- **平台对比：** 各平台发布数、浏览量、平均互动
- **趋势图表：** 日期维度的发布量和浏览量变化
- **热门内容：** 表现最好的内容排行

---

### 7. 日程管理

- 创建、编辑、完成、取消日程
- AI 自动从对话中识别日程信息
- 按时间排序展示（即将到来 + 最近过去）
- 支持标题、描述、时间、地点

## AI 能力总览

| 能力 | 说明 |
|------|------|
| **文字生成** | DeepSeek / Qwen 等模型，支持文案、摘要、改写、分析 |
| **图片理解** | 视觉模型识别图片内容、OCR 提取文字 |
| **视频理解** | 视频语音识别 + 内容分析 |
| **图片生成** | AI 生图（Doubao Seedream），多种风格和比例 |
| **图片编辑** | 去背景、画质增强、智能扩图 |
| **视频生成** | 文字转视频（Wan / Seedance），自动字幕、BGM |
| **语音合成** | CosyVoice / Kokoro TTS，多音色 |
| **数字人** | 照片生成形象 + 口播动画 |
| **联网搜索** | 聚合搜索（百度 / Google / Bing），获取实时信息 |
| **链接抓取** | jina.ai Reader 提取文章正文 |
