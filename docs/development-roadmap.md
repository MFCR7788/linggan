# 灵集 - 分阶段开发计划

## 📋 总体开发路线图

### 阶段一：前端API集成（当前阶段）
目标：将现有页面的mock数据替换为真实API调用

### 阶段二：完善API
目标：补充所有剩余的API端点

### 阶段三：AI服务集成
目标：接入DeepSeek、豆包等AI服务，实现智能分析功能

---

## 🎯 阶段一：前端API集成

### 1.1 创建前端API客户端
- 创建统一的API请求封装
- 实现请求/响应拦截器
- 添加错误处理和重试机制

### 1.2 创建React Query / SWR Hooks
- 实现数据获取和缓存
- 添加乐观更新
- 实现自动重新验证

### 1.3 更新页面组件
- **登录页面**：集成真实用户认证
- **首页**：获取真实灵感列表和热点数据
- **灵感库**：集成分类、标签、内容CRUD
- **灵感详情**：获取真实内容数据
- **AI创作**：集成AI调用
- **个人中心**：获取用户数据

### 1.4 完善用户体验
- 添加加载状态（skeleton、loading spinner）
- 添加错误边界
- 添加空状态UI
- 实现下拉刷新和上拉加载更多

- 优化索引和查询

## 🚀 立即开始

### 第一个任务：创建前端API客户端

**目标：** 创建一个统一的API请求工具

**文件：**
- `/src/lib/api-client.ts` - API客户端基础封装
- `/src/hooks/use-api.ts` - React Query hooks
- `/src/providers/api-provider.tsx` - 全局API提供者

**功能：**
- 统一的请求/响应处理
- 认证token管理
- 错误处理和重试
- 类型安全的API调用
