# 灵集 — 开发步骤规划

> 文档日期：2026-05-20
> 开发阶段：第一阶段（MVP）

---

## 一、开发阶段规划

### 第一阶段（MVP）：4-6 周

| 周次 | 模块 | 核心任务 |
|-----|------|---------|
| 第 1 周 | 项目初始化 + 用户系统 | Next.js 项目、Supabase 配置、登录/注册页面 |
| 第 2 周 | 灵感记录核心 | 文字/语音/链接/图片/视频上传 + AI 总结 |
| 第 3 周 | 灵感库 + 搜索 | 列表/详情页、筛选、搜索、标签分类 |
| 第 4 周 | AI 写作助手 | 素材选择、生成类型选择、图文并茂生成、去 AI 味 |
| 第 5 周 | 热点监控 + 通知 | 关键词配置、热点库、实时通知 |
| 第 6 周 | 订阅 + 协作 | 订阅页面、用量限制、协作空间 |

---

## 二、UI 设计优先级

### 第一批（核心）：v0.app 先设计
1. **登录/注册页** - 手机验证码登录
2. **首页** - 快速记录入口 + 最近灵感
3. **灵感库列表页** - 浏览/筛选/搜索
4. **灵感详情页** - 查看/编辑/标记/生成内容
5. **AI 写作页** - 选择素材 → 生成内容

### 第二批：
6. **热点监控配置页** - 添加关键词
7. **热点库列表页** - 浏览热点
8. **热点详情页** - 查看详情/转灵感
9. **个人中心页** - 个人信息/用量/订阅
10. **订阅页** - 套餐选择

---

## 三、技术实现顺序

### Step 1：基础设施
- Next.js 14 项目初始化
- Tailwind CSS 配置
- Supabase 客户端配置
- 基础 UI 组件库（shadcn/ui）
- 路由配置

### Step 2：用户认证
- Supabase Auth 集成
- 登录/注册页面
- 会话管理
- 用户信息页

### Step 3：灵感记录
- 文字记录表单
- 图片/视频上传（Supabase Storage）
- 链接解析输入框
- AI 处理状态展示
- 保存到数据库

### Step 4：灵感库
- 列表页（时间/热度/分类排序）
- 筛选器（平台/标签/时间）
- 搜索框
- 详情页

### Step 5：AI 写作
- 素材选择（多选灵感/热点）
- 生成类型选择（小红书/短视频/公众号）
- 文风选择（去 AI 味）
- 生成结果展示 + 编辑
- 配图展示 + 替换

### Step 6：热点监控
- 关键词配置页
- 热点列表页
- 热点详情页
- 通知中心

### Step 7：订阅系统
- 套餐展示页
- 用量进度显示
- 支付集成（第一阶段可先模拟）

---

## 四、设计规范

### 视觉风格
- 主色：紫色 #7C3AED（渐变到蓝色）
- 风格：简洁、现代、中文友好
- 设计理念：简便、易上手、少操作

### 响应式优先
- 移动端 H5：iPhone SE → iPad
- Web 端：13-inch → 27-inch
- 同一套代码库，响应式适配
