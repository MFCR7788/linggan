## 测试准备

### 前置条件

- [ ] 已注册灵集账号（或使用开发环境 dev auth）
- [ ] 账号有足够灵力点数（至少 50 灵力，可在 `/profile/billing` 查看）
- [ ] 浏览器已清除缓存 / 使用无痕模式开始
- [ ] 网络通畅，API 域名可访问

### 开发环境配置 (local)

```bash
# .env.local 需要包含 SUPABASE_ANON_KEY
cp .env.example .env.local
# 编辑 .env.local 填入实际值
```

### Dev Auth 快捷登录 (仅 localhost / 配置了 DEV_AUTH_SECRET)

1. 浏览器控制台执行：
```js
localStorage.setItem('dev_user', JSON.stringify({ id: '你的测试用户UUID' }));
location.reload();
```
2. 自动绕过 Supabase Auth 登录

## 附录 C：已知限制与说明

1. **AI 视频生成**耗时较长 (数分钟)，测试时需耐心等待，可观察任务进度
2. **数字人**需要预先生成/选择形象，首次使用需创建
3. **平台发布**需要先完成 OAuth 授权 (`/profile/integrations`)
4. **语音录制**需要浏览器 HTTPS 或 localhost 环境
5. **声音克隆**需要上传 10-30 秒清晰人声音频作为参考
6. **Dev Auth** 仅在 localhost 或配置 `DEV_AUTH_SECRET` 时可用
7. **微信支付**需在微信内打开或使用微信 H5 支付
8. 部分 AI 模型在 Vercel Serverless 下有 60s 超时限制 (视频类接口在 `vercel.json` 中配置了 `maxDuration: 60s`)
