## 🎨 设计规范统一前置

**所有页面共用（玻璃拟态科技风格 + 模块化设计）：**
1. **整体底色**：采用深邃科技渐变背景，从`#0A1629`渐变至`#1A365D`，添加微弱、不突兀的星空粒子纹理（低透明度，不遮挡内容），营造轻盈高级的科技氛围
2. **卡片/模块设计**：统一改为半透明白色玻璃效果，背景色设置为`rgba(255,255,255,0.12)`，搭配`1px`细白色描边，开启毛玻璃模糊效果（模糊度`8-10px`），圆角统一为`16px`，无厚重阴影，仅添加轻微柔光投影
3. **配色规范**：主色调统一为清透浅蓝色`#3B82F6`，激活状态的按钮、高亮数据、选中标签均用该色，且添加轻微霓虹发光效果（发光强度适中，不刺眼）；辅助色保持原有逻辑（紧急状态`#EF4444`、正常状态`#22C55E`），均添加轻微柔光，适配玻璃拟态质感
4. **文字规范**：统一使用现代无衬线字体，标题字重`600`、颜色`#FFFFFF`，正文字重`400`、颜色`#E5E7EB`，关键数据（如总热点、今日新增）用主色高亮，字体大小层级分明，确保在玻璃背景上清晰可读，无模糊感
5. **图标与交互**：所有图标统一改为极简线性风格，颜色与文字颜色一致，添加轻微外发光效果；按钮hover态添加玻璃反光效果，点击态轻微加深透明度，交互反馈柔和不突兀
6. **细节统一**：所有模块内边距统一，标签、按钮圆角与卡片圆角适配（标签圆角`8px`），热点列表项、筛选栏、导航栏均遵循上述玻璃拟态规范，无任何违和感，整体呈现通透、高级、统一的科技时尚感
7. **模块化设计原则**：
   - 所有功能模块独立设计，可单独复用
   - 统一组件库（按钮、卡片、输入框、标签等），确保视觉一致性
   - 功能模块可自由组合，便于后续扩展
   - 状态管理统一规范（加载、空状态、错误状态）
8. **用户便捷操作原则**：
   - 一步完成原则：能在一步完成的绝不拆分为多步
   - 智能预判：系统预判用户可能的操作，提供快捷入口
   - 上下文延续：从一个操作到下一个，保持上下文信息
   - 必要流程保留：核心流程（如创作）保留完整步骤，但可通过快捷入口快速体验
- 响应式：移动端优先，自适应 PC
- 设计理念：简便、易上手、操作少
- 技术栈：Next.js + React + Tailwind CSS + shadcn/ui
